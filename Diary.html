22.3.27

Key words: Matlab 2018, Coalesce

Descrption

There would be errors like:<未定义与 'DivideBinaryOperatorNode' 类型的输入参数相对应的函数 'asin'>, if the latter function is not supported in the used package.

Solution

Try to add a function to explain and call the latter function inside the package
1. create asin function in the "\coalesce\symbolic\@ExpressionTree" folder
2. uncomment the INVERSE TRIGONOMETRIC OPERATIONS parts at line 350 to use asin fuction.
in file ExpressionNode.m
3. create AsinUnaryOperatorNode.m in the "\coalesce\symbolic" folder
4. add asin part in the "\coalesce\symbolic\@ExpressionTree\char.m" file
5. add asin part in the "\coalesce\symbolic\@ExpressionTree\diff.m" file
6. add asin part in the "\coalesce\symbolic\@ExpressionTree\eval.m" file
7. add asin part in the "\coalesce\symbolic\@ExpressionTree\private\simplify_.m" file
8. small note, this package use ./ or .* everywhere instead of / or *
